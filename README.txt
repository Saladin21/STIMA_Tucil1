Program ini berfungsi untuk menyelesaikan persoalan Cryptarithmetic dengan jumlah huruf unik maksimal 10. Untuk menjalankan program diperlukan Python 3.

Cara menggunakan program:
1. Buat file txt berisi persoalan cryptoarithmetic pada folder test dengan format berikut.
 COCA
 COLA+
-------
OASIS
2. Jalankan program dengan cara masuk ke folder bin, lalu jalankan Cryptarithmetic
3. Masukkan nama file yang telah dibuat sebelumnya di folder test. (contoh: file.txt)
4. Jika terdapat solusi, soal dan solusi akan ditampilkan di layar.
